# QRFlow

> QR code generator and scanner — create and decode QR codes in seconds.

## Features

- Generate QR codes from any text, URL, or contact info
- Scan QR codes using your device camera in real time
- Customize QR colors, add logos, and set error correction levels
- Choose from multiple output formats: PNG, SVG, JPEG
- Decode QR codes from uploaded image files
- Generate vCard and WiFi network QR codes
- Adjust module size and margin padding
- Batch generate multiple QR codes at once
- Download individual codes or all as a ZIP archive

## Why QRFlow?

QRFlow processes everything on the client side using lightweight JavaScript libraries. No QR code data is ever sent to a server — your links, contacts, and WiFi credentials remain private. It's free, works offline, and is the quickest way to generate or scan QR codes without installing an app.
